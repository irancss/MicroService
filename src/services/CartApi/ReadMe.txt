11. میکروسرویس سبد خرید (Cart Service)
وظایف:
مدیریت سبد خرید کاربران (افزودن، حذف، تغییر تعداد).
اعمال تخفیف‌ها و محاسبه قیمت نهایی.
ذخیره موقت سبد برای کاربران مهمان.
ادغام با موجودی و سفارشات.
دیتابیس پیشنهادی:
Redis: برای سرعت و ذخیره‌سازی موقت.
زبان/فریمورک:
.NET Core: با StackExchange.Redis.
جایگزین پیشنهادی: Node.js برای عملکرد سبک‌تر.
تکنولوژی‌های کمکی:
Memcached: به‌عنوان جایگزین Redis در برخی سناریوها.
Kafka: برای همگام‌سازی سبد با سرویس سفارشات.
