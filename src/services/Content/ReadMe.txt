15. میکروسرویس مدیریت محتوای استاتیک (Content Service) (اضافه شده برای حرفه‌ای‌تر شدن)
وظایف:
مدیریت صفحات فرود، بلاگ، و محتوای تبلیغاتی.
پشتیبانی از چندزبانگی (Localization).
بهینه‌سازی SEO برای محتوا.
دیتابیس پیشنهادی:
MongoDB: برای انعطاف‌پذیری محتوا.
S3: برای ذخیره‌سازی فایل‌ها.
زبان/فریمورک:
.NET Core: با Razor Pages.
جایگزین پیشنهادی: Node.js با Next.js برای SSR و SEO.
تکنولوژی‌های کمکی:
Cloudflare Workers: برای تحویل سریع محتوا.
Strapi: برای CMS بدون سر (Headless CMS).
