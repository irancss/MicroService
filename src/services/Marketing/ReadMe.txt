13. میکروسرویس بازاریابی (Marketing Service)
وظایف:
مدیریت کمپین‌های تبلیغاتی هدفمند.
تولید و اعتبارسنجی کوپن‌ها و کدهای تخفیف.
تحلیل ROI و اثربخشی کمپین‌ها.
ادغام با سیستم توصیه‌گر برای تبلیغات شخصی‌سازی‌شده.
دیتابیس پیشنهادی:
PostgreSQL: برای روابط پیچیده و گزارش‌گیری.
زبان/فریمورک:
.NET Core: با Entity Framework Core.
جایگزین پیشنهادی: Python برای ادغام با ابزارهای تحلیلی.
تکنولوژی‌های کمکی:
Segment: برای ردیابی کمپین‌ها.
Redis: برای کش کردن کدهای تخفیف.
